# QSingleLineEdit
Example application to show how I think Qt's text selection could be improved for single-line text input fields.

The idea is that when people wants to select the beginning (left part) of a text string, they would probably move the mouse to the left. And to select the end (right part) they would probably move the mouse to the right side.

However, some GUI frameworks treat all text input fields as a multi-line which means you have to move the mouse up/down to do a left/right text selection.
